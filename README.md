# OpenComputers CE: Ender Cable

OpenComputers CE: Ender Cable is an addon for OpenComputers CE 1.20.1.

OpenComputers by Sangar, Vexatos, payonel, magik6k, Lord Joda, Github Contributors and modern port by akki697222.

This addon adds an Ender-style cable for OpenComputers CE networks. It is intended to bring back the old OpenComputers conduit idea from the Ender IO 1.12.2 era as a standalone mod, without replacing, patching, or depending on Ender IO itself.

## Status

This repository starts with a minimal Forge 1.20.1 implementation:

- `occe_ender_cable:ender_cable`
- OpenComputers CE `Environment` and `SidedEnvironment` capability support
- basic network joining and persistence
- dynamic six-sided connection blockstates for adjacent OpenComputers network nodes
- standalone cable block and item model
- animated old OpenComputers conduit-inspired textures
- handheld OpenComputers network visualizer overlay for loaded nearby nodes

Planned follow-up work:

- facade support
- optional Ender IO facade compatibility
- optional Applied Energistics 2 facade compatibility
- a separate OpenComputers CE: AE2 Integration addon for ME network inventory queries
